# 哔哩哔哩（bilibili.com）播放器调整

一个B站的用户脚本，调整B站播放器设置，增加一些实用的功能。

##  脚本功能
* 自定义功能快捷键。
* 自定义播放器大小，定位到播放器，自动隐藏播放器控制栏。
* 更改默认的播放模式（宽屏模式，全屏模式，网页全屏模式）。
* 更改默认的弹幕显示方式，自动关灯。
* 自动播放视频，自动播放下一个视频，循环播放，从指定时间开始播放，更改视频默认播放速度，双击全屏。
